autointent.custom_types.TASK_TYPES¶

Represents the supported task types in AutoIntent: - “multiclass”: Single-label classification. - “multilabel”: Multi-label classification. - “multiclass_as_multilabel”: Multi-class treated as multi-label.

autointent.custom_types.WEIGHT_TYPES¶

Represents weight calculation methods: - “uniform”: Equal weight for all elements. - “distance”: Weights based on distance. - “closest”: Prioritizes closest elements.
